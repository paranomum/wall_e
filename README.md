# wall_e?

> У вас мусор на беке, а сваггер - шляпа

Генератор на основе openapi-generator - open source проекта генерации api

# Генерация с помощью wall_e

## config-service.json

```
{
  "basePath" : "artifact_id",
  "services" : {
        "service_name" : "service_url"
  }
}
```

## generate

```
java -jar cli.jar generate \
    -c ${path_to_config-service.json} \
    -s ${service_name_from_config-service.json} \
    -o ${output_directory}
```
